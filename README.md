### Run in Docker

```bash
docker run --rm -d \
  --name curexcratebot \
  -e CREATOR_ID=12345 \
  -e BOT_TOKEN=123:xxx \
  -e BOT_USERNAME=name_bot \
  ghcr.io/helltar/curexcratebot:latest
```

### Commands

- `/cur` (Example: `/cur 1 usd to uah`)
- `/privacy`
